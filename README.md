# nukopy's dotfiles repository

Dotfiles repository for OSX(macOS) to get my work environment ready in just a few moments.

## Start

```bash
$ git clone http://github.com/nukopy/dotfiles.git
$ cd dotfiles
$ source make_symlink.sh
```
